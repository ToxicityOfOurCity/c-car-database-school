# Βάση δεδομένων αυτοκινήτων σε C.

Αυτό είναι ένα «απλό» (800 γραμμές λένε το αντίθετο) πρόγραμμα σε C. Αρχικά ήταν ένα σχολικό project, αλλά τελικά μου πήρε τόσο πολύ χρόνο που αποφάσισα να το ανεβάσω εδώ.
Κάτι που αξίζει να αναφερθεί είναι ότι γνωρίζω πως το πρόγραμμα είναι γραμμένο σε **ακραία αρχάριο στυλ**, τόσο στους αλγορίθμους όσο και στη σύνταξη, όμως αυτό ήταν και οδηγία της σχολής: να ΜΗΝ το παρακάνουμε.
Για παράδειγμα, το bubble sort ήταν η μοναδική μέθοδος ταξινόμησης που επιτρεπόταν να χρησιμοποιήσω.

Το πρόγραμμα αποτελείται από πολλά πράγματα. Δεν θα τα απαριθμήσω όλα ΕΔΩ, αλλά είστε ευπρόσδεκτοι να χρησιμοποιήσετε τον πηγαίο κώδικα, να τον κάνετε compile και να το δοκιμάσετε.

# Κεντρικό Μενού

Με το άνοιγμα του .exe, ο χρήστης βλέπει το παρακάτω κεντρικό μενού:

![mainmenu](/images/main_menu.png)

Και του παρουσιάζονται 7 επιλογές.

 1. Εμφάνιση ολόκληρης της βάσης δεδομένων cars.txt.

 2. Εισαγωγή νέου αυτοκινήτου στη βάση δεδομένων. <sup>(δεν εξηγείται για να μη καταλαμβάνει πολύ χώρο)</sup>

 3. Διαγραφή αυτοκινήτου από τη βάση δεδομένων. <sup>(δεν εξηγείται για να μη καταλαμβάνει πολύ χώρο)</sup>

 4. Ενημέρωση υπάρχοντος αυτοκινήτου στη βάση δεδομένων.

 5. Αναζήτηση αυτοκινήτων στη βάση δεδομένων.

 6. Χειροκίνητη αποθήκευση της βάσης δεδομένων (επίσης απαίτηση της εργασίας, όχι δική μου επιλογή).

 0. Έξοδος από το πρόγραμμα.
